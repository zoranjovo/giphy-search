## Running locally
1. Clone the repo
2. Install dependencies: `npm install`
3. Run the react dev server: `npm run start`

## Deploying to AWS Cloudfront
1. Set up AWS profile: `npm run aws:profile`
2. Deploy: `npm run deploy:staging`
3. Sit back and let baseline deploy it for you
